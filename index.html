Syntactically Awesome Style Sheets

You can install and use Sass by implemeting two different methods.

  1. Installing via NPM
  2. Using the VS Code extension Live Sass Compiler

My current preferred method is #2, using the VS Code extension. Once the extension is installed, you'll find a new option at the bottom on VS Code's UI that says 'Watch Sass'. Click on the button to compile your .scss files into a single .css file.
